Setup Guide

How to Connect to Google Sheets

Follow these steps to connect your Student Management System to Google Sheets:

1

Create Google Apps Script

Go to script.google.com and create a new project.

2

Paste Backend Code

Replace the default code with the backend script provided earlier.

3

Deploy as Web App

Click "Deploy" > "New deployment", select "Web app", set access to "Anyone", and deploy.

4

Copy Web App URL

After deployment, copy the web app URL provided by Google.

https://script.google.com/macros/s/.../exec
5

Paste URL in System

Go to the Google Sheets tab in this system and paste the URL in the "Google Apps Script URL" field.

6

Connect to Sheets

Click "Connect to Sheets" and your system will be connected to Google Sheets!

Backend API Endpoints

Your Google Apps Script will handle these actions:

// Get all students
GET your-script-url?action=getStudents

// Add a new student
POST your-script-url?action=addStudent
{ studentId: "S001", firstName: "John", ... }

// Update a student
POST your-script-url?action=updateStudent
{ studentId: "S001", firstName: "John Updated", ... }

// Delete a student
POST your-script-url?action=deleteStudent
{ studentId: "S001" }
