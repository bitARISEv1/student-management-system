Deployment Guide

1 Deploy to GitHub Pages

Host your application for free using GitHub Pages:

  1. Create a GitHub account if you don't have one
  2. Create a new repository named "student-management-system"
  3. Upload all your files to the repository
  4. Go to repository Settings > Pages
  5. Set source to "Deploy from branch" and select main branch
  6. Your site will be published at: https://[username].github.io/student-management-system
# Steps to deploy to GitHub Pages
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/[username]/student-management-system.git
git push -u origin main

2 Deploy to Netlify

For more advanced features, deploy to Netlify:

  1. Create a Netlify account (free)
  2. Connect to your GitHub repository
  3. Select your repository and branch
  4. Set build command to empty (since it's a static site)
  5. Set publish directory to "/"
  6. Click "Deploy site"
  7. Your site will be published at a custom netlify.app domain

Netlify provides continuous deployment - every time you push to GitHub, your site will automatically update.

3 Connect to Google Sheets

Follow these steps to connect your application to Google Sheets:

  1. Create a new Google Sheet named "Student Management System"
  2. Create worksheets named "Students", "Courses", and "Enrollments"
  3. Set up headers as shown below
  4. Publish your sheet to the web (File > Share > Publish to web)
  5. Copy the Sheet ID from the URL
  6. Paste the Sheet ID in the Google Sheets Setup section of this app
